# Computational Network Medicine Lab

Website for the **Computational Network Medicine Lab**, College of Korean Medicine, Wonkwang University (원광대학교 한의과대학).

**Live site:** https://wonyung-lee.github.io

Research areas: network pharmacology, graph-based machine learning (GNN / knowledge graphs), herb–drug interaction prediction, and LLM applications for Korean medicine.

## Local preview

Requires [Docker](https://www.docker.com/products/docker-desktop/).

```bash
docker compose up
```

Then open <http://localhost:8080>.

## Deploy

Push to the `main` branch — GitHub Actions builds the site and publishes it automatically.
